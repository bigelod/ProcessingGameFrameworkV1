This folder contains sounds you can use as part of your games (public domain!). Copy the files from this folder into the root folder and then import them in the code.

For more resources, go to opengameart.org and freesound.org ! Remember to check the license as to whether credit is needed (not public domain) and if it can be used commercially (meaning you can sell games made with those graphics) if you plan to make a commercial game!
